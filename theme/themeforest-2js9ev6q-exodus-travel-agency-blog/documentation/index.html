Style Variations

The class color is related to the background color of any element. eg:. dark

To remove the background color of any element. eg:. no-bg

  • Adding shadow will add a shadow to the element. eg:. shadow
  • Adding b- (for border) will color the border of the element. eg:. b-dark
  • Adding t- (for text) will color the text of the element. eg:. t-dark

Example:

  • dark t-white
  • pine t-snow
  • b-brick t-brick
  • I am a red! t-red
  • I am a Vibrant Blue t-blue-vibrant

To apply a border only at the top or the bottom, work with the class variation border-b as in border bottom.

To modify the border weight, work with the class variation border-t-2 as in border top 2px

Do not add the border class as it will apply a border on all sides.


To recap:

  • Adding -t or -b will add a the respective border top/bottom. eg:. border-t
  • Adding -2 will increase the border size to 2px. eg:. border-t-2

Example:

  • b-dark border-t
  • b-pine border-b-2

Button styles

There is 1 type of button primary-btn. But with AZ Styles you can customize it in the most flexible and easy way.

Full flexibility.

  • Adding round will add a border radius of 50px. eg:. primary-btn round
  • Adding square will remove any border radius making the button square. eg:. primary-btn square
  • Adding small will reduce the button size. eg:. primary-btn small
  • Adding large will increase the button size. eg:. primary-btn large

Example:

  • primary-btn small dark t-snow
  • primary-btn pine t-white
  • primary-btn large shadow light-sky t-dark
  • primary-line-btn b-salt t-salt
  • primary-line-btn round b-light-moon t-light-moon
  • primary-btn square viola t-lavender

Margin & Padding

You can apply a margin or a padding to a specific element, div or section by using the class p & m + the value. eg:. m-50 or p-50

It works for both (padding and margin).

  • Available values for margin are m-20 m-50 m-100 m-150 m-200.
  • Available values for margin-top are mt-20 mt-50 mt-100 mt-150 mt-200.
  • Available values for margin-bottom are mb-20 mb-50 mb-100 mb-150 mb-200.

You can also remove margin top, left & right by using mt & -minus + the value. eg:. mt-minus-50 (only for margin)

  • Available values for mt, ml & mr are -20 -50 -100 -150 -200 -250 -300. eg:. mt-minus-50

Opacity

You can apply a opacity to a specific element, div or section by using the class op- + the value. eg:. op-2 opacity 20%
